# KrishiSaarthi - Empowering Farmers with AI

KrishiSaarthi is a professional, end-to-end agriculture platform designed to support farmers with AI-powered insights, a functional marketplace, and a community for knowledge sharing.

## 🚀 Features

- **AI Soil Analysis:** Advanced ML integration for soil health prediction and fertilizer recommendations.
- **Marketplace:** A professional commerce system for farmers to buy and sell agricultural products.
- **Community Portal:** A platform for farmers to connect, share expertise, and get support.
- **Farmer Dashboard:** A unified interface for tracking activities and accessing all tools.

## 🏗️ Project Structure

The project is divided into two main parts:

- **[`frontend/`](./frontend)**: A modern, responsive React + Vite application.
- **[`backend/`](./backend)**: A robust Django + MongoDB backend handling AI logic and marketplace operations.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS (or modern Vanilla CSS).
- **Backend:** Python (Django), MongoDB, PyTesseract (OCR), Scikit-Learn (ML).
- **Authentication:** JWT-based secure authentication system.

## 🏁 Getting Started

### Prerequisites
- Node.js & npm
- Python 3.8+
- MongoDB instance

### Backend Setup
1. `cd backend`
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)
4. Install dependencies: `pip install -r requirements.txt`
5. Configure `.env` based on `.env.example`.
6. Run: `python manage.py runserver`

### Frontend Setup
1. `cd frontend`
2. Install: `npm install`
3. Start: `npm run dev`
